# Atividade Avaliativa - Aplicativo de Clínica com SQLite (React Native)

## Objetivo

Desenvolver um aplicativo mobile para gerenciamento de pacientes e consultas de uma
clínica utilizando SQLite.


### Funcionalidades exigidas:

- CRUD completo para Pacientes
- CRUD completo para Consultas
- Tela de listagem, cadastro, edição e exclusão
- Relacionamento entre Pacientes e Consultas

### Entregáveis:

- Código-fonte completo
- Projeto organizado com SQLite integrado

### Critérios de Avaliação:

- CRUD Pacientes
- CRUD Consultas
- Banco SQLite
- Interface/Navegação
- Qualidade do código
